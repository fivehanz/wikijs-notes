# What?
* main sole purpose of computer is to **execute programs**

# Basic Elements
* **Processor (CPU):**
	* heart, brain of the computer
	* all programs are executed by processor
* **Main memory:**
	* store information
* **I/O modules:**
	* input -> computer -> output
* **System bus:**
	* connects all other parts
